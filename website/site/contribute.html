How to contribute to the register of releases

There are two ways to contribute to this register of releases:

  1. Send an email about a single release to {{config.contact_email}} (this link will open with a pre-populated template for you to complete).

  2. Send an email about multiple releases to {{config.contact_email}} and attach a CSV file in this format.

  3. Edit the csv files in the data/ directory of this GitHub repository.

    To add a release history for a new product or service, you can create a new csv file in the data/ directory and add a new resource to the datapackage.json file.
